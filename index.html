📋 구글 시트 연결 방법 (Apps Script)
  1. 가이드를 기록할 구글 스프레드시트를 열고, B1에 #S, C1에 내용 헤더를 입력
  2. 상단 메뉴 → 확장 프로그램 → Apps Script 클릭
  3. 기본 Code.gs 파일의 내용을 모두 지우고, 아래 코드를 붙여넣기:
    → guide-apps-script.js 파일 열기 → 전체 복사 → Code.gs에 붙여넣기
  4. 상단 배포 → 새 배포 클릭
    · 유형: 웹 앱 선택
    · 다음 계정으로 실행: 나
    · 액세스 권한: 모든 사용자
  5. 배포 클릭 → 나오는 웹 앱 URL(https://script.google.com/macros/s/...)을 복사
  6. 위 Apps Script URL 칸에 붙여넣기 → 왼쪽 초록 점이 켜지면 연결 완료!
